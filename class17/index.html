Research Methods for Global Studies

Class 17: Oral Presentations

John Palmer

Key challenges

  • overcoming fear
  • communicating clearly
  • engaging audience

Preparation

  • Start with messy ideas
  • Distill into outline
  • Write out some or all of speech - especially key phrasing
  • Edit
  • Distill speech to short outline/reminders
  • Practice

Implementation

  • dress
  • movement
  • timing
  • engaging the audience
  • managing slides

Slides: Core Considerations

  • Slides should aid talk, not other way around
  • Slides should not distract
  • Slides should not be the default

Slides: Technical Considerations

  • Use clear, sans serif font
  • Keep them simple
  • Less is better
  • Make things big enough to read
